# 🤖 Cohort 20 Telegram Bot

A student-led Telegram Bot built entirely with JavaScript by learners of **CIL Academy Cohort 20**.

The goal of this project is not only to build a useful Telegram bot for our unofficial community but also to transform our Module 1 knowledge into practical software engineering experience.

---

## 🎯 Objectives

- Practice JavaScript
- Learn Git & GitHub collaboration
- Apply the Software Development Life Cycle (SDLC)
- Experience Agile teamwork
- Deploy a real cloud application
- Build something valuable for our cohort

---

## ✨ Features

- 👋 Welcome new members
- 🔍 Search study resources
- ❓ Frequently Asked Questions
- 🚨 Automated reminders
- 📢 Broadcast announcements
- 👤 Admin management

---

## 🛠 Technology Stack

- JavaScript (Node.js)
- Telegram Bot API
- Supabase
- Render
- Git & GitHub

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/CIL-Cohort-20/Telegram-BOT.git
```

Install dependencies

```bash
npm install
```

Start the bot

```bash
npm start
```
