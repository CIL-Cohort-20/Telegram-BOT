# Code of Conduct

This project is a learning environment.

Be respectful.

Be patient.

Be willing to help.

Everyone here is learning.

---

## We Encourage

- Asking questions
- Helping others
- Sharing knowledge
- Constructive feedback
- Respectful discussions

---

## We Don't Encourage

- Harassment
- Insults
- Toxic behaviour
- Discrimination
- Plagiarism
- Public shaming

---

## Remember

Today's beginner may become tomorrow's mentor.

Let's build something amazing together.
