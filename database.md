# 🗄️ Telegram Bot Database

This directory contains the database design for the **CIL Cohort 20 Telegram Bot**.

The backend is powered by **Supabase (PostgreSQL)**.

---

## 🎯 Purpose

The database stores all information required by the Telegram Bot, including:

- Users
- Learning Resources
- Frequently Asked Questions (FAQ)
- Reminders
- Announcements
- Administrators

---

## 🛠 Technology

- Supabase
- PostgreSQL
- SQL
- Row Level Security (RLS)

---

## Database Tables

| Table | Purpose |
|--------|---------|
| users | Stores Telegram users |
| resources | Stores learning resources |
| faq | Stores frequently asked questions |
| reminders | Stores scheduled reminders |
| announcements | Stores broadcast messages |
| admins | Stores bot administrators |
